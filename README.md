# 📦 Módulo de Inventario

Una aplicación web simple y eficiente para la gestión de inventarios, desarrollada con HTML y JavaScript vanilla. Permite gestionar productos, monitorear stock y recibir alertas automáticas cuando los niveles están bajos.

## 🚀 Demo en Vivo

🌐 **[Ver Aplicación](https://modulo-inventario-hrxqzblns-luisernestosalas-2775s-projects.vercel.app)**

## ✨ Características

- ✅ **Gestión de Productos**: Agregar, editar y eliminar productos
- 💰 **Control de Precios**: Registro y actualización de precios
- 📊 **Monitoreo de Stock**: Seguimiento en tiempo real del inventario
- 🚨 **Alertas Inteligentes**: Notificaciones automáticas para stock bajo
- 💾 **Persistencia Local**: Los datos se guardan en el navegador
- 📱 **Diseño Responsivo**: Compatible con dispositivos móviles y desktop
- ⚡ **Rendimiento Optimizado**: Sin dependencias externas, carga ultrarrápida
